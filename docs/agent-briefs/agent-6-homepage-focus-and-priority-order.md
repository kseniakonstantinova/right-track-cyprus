# Agent 6 Brief: Refocus Homepage on Core Clinic Intent

## Role
You are responsible only for improving the homepage so it behaves more clearly as the main commercial landing page for the clinic.

This is a homepage content hierarchy task.

## File Ownership
You may edit only:
- `index.html`
- `index-el.html`

You must not edit:
- service pages
- blog articles
- community pages
- event pages
- docs

## Background
The homepage currently mixes several intents near the top:
- clinic homepage
- seminar/community promotion
- survey/research promotion
- padel camp promotion
- Pilates launch

That makes the page feel less focused on the primary job:
- establish Right Track as the physiotherapy clinic in Nicosia / Strovolos
- support brand queries
- support local commercial queries
- drive booking

## Core Goal
Make the homepage clearly communicate, above all else:
- who the clinic is
- where it is
- what core services it provides
- that it accepts GESY
- how to book

Secondary initiatives can remain, but must not dominate the first screen or interrupt the main commercial path.

## Required Work

### 1. Re-evaluate above-the-fold hierarchy
The first screen should prioritize:
- clinic identity
- location
- physiotherapy / rehab / sports physio relevance
- GESY trust signal
- primary CTA to booking

### 2. Demote non-core promotional content
Seminar, survey, camp, and other campaign content can stay, but should move lower if they currently compete with the main offer.

### 3. Strengthen internal pathways
Above the fold or soon after, make it easy to reach:
- booking
- core service pages
- GESY section / GESY articles
- key clinician profiles

### 4. Preserve brand tone
The homepage should still feel like Right Track:
- athletic
- clinical
- performance-oriented

Do not make it generic or wellness-studio-like.

## Constraints
- do not change URL
- do not rewrite the entire site
- do not create new pages
- do not edit blog or service files
- keep EN/EL parity in intent and structure

## Acceptance Criteria
Your work is done when:
- the homepage top section is clearly clinic-first
- the primary CTA is obvious
- local and commercial relevance are stronger
- secondary campaigns no longer dilute the main intent

## Suggested Final Report
When done, report:
- what changed in the homepage hierarchy
- which promotional elements were demoted or reframed
- how booking / services / GESY access became clearer
