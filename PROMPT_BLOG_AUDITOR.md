**Option A: Inline Contextual Citations (Recommended for this blog)**

## HOW TO USE CONSENSUS (consensus.app)

When verifying claims or finding sources:

1. Search Consensus for the specific clinical claim (e.g., "ACL reconstruction return to sport timing reinjury rate")
2. Review the synthesized answer AND the underlying papers
3. Always cite the **original published paper**, not Consensus
4. Use Consensus to check if there are **more recent or higher-impact papers** than the ones commonly cited
5. If Consensus shows a scientific consensus meter, note the confidence level (e.g., "Strong consensus: Yes, supported by 9/10 studies")
